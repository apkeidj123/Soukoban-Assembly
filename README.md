# Soukoban-Assembly

Game introduction:

```
Move locked boxes on to magic squares to make them open.
If all boxes are open, go to the next level.
There are 50 levels.

```

Keystroke/Option illustration:

```

Move: up down left right -> W S A D
ESC: menu:
(1)restart
(2)next stage
(3)to the title
(4)exit
ENTER: confirm/
R: restart
m: turn on/turn off the BGM
`: cheat key

```
